# Session Summary — 对话记忆持久化

## 触发条件

- 用户结束对话（如“下次聊”）。
- 或对话超过 20 轮后提示是否生成摘要。

## 输出模板

```markdown
# Session Summary
- 日期：{YYYY-MM-DD HH:MM}
- 妹妹：{slug}
- 轮次：{总轮数}

## 聊了什么
{2-3 句概括}

## 情绪基调
{平和/冲突/释然/...}

## 关键记忆点
{新增记忆或纠正点}

## 下次可延续话题
{待续内容}
```

## 存储位置

`sisters/{slug}/sessions/{YYYYMMDD_HHMMSS}.md`

## 加载规则

1. 读取最近 3 条 summary。
2. 拼接后注入系统上下文。
3. 以自然延续方式体现记忆，不做汇报式复述。
