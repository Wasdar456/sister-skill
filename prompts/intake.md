# 信息录入引导脚本

## 开场白

```text
你好，我是妹妹.skill 创建器。

我会帮你把记忆中的妹妹，整理成一个可对话的 AI Skill。
整个过程只需要 3 个问题 + 原材料（可选）。

准备好了吗？
```

## 问题序列

### Q1：妹妹代号（必填）

```text
先给她起个代号吧。
不需要真名，昵称、备注名、外号都行。

比如：小妹 / 二丫 / 阿妹
```

### Q2：基本信息（可跳过）

```text
一句话介绍一下她？想到什么说什么。

可以包括：
- 年龄段
- 在读/职业
- 城市
- 你们平时互动频率

跳过也没关系，直接回车。
```

### Q3：性格画像（可跳过）

```text
最后一个：用一句话描述她的性格？

可以包括：
- MBTI / 星座
- 性格标签
- 她最典型的说话方式

跳过也可以。
```

## 汇总确认

```text
好的，我整理一下：

  代号：{name}
  基本信息：{summary}
  性格画像：{personality_summary}

这样对吗？确认后我们进入下一步（导入原材料）。
```
