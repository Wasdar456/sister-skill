# 对话纠正处理器

## 任务

当用户在使用妹妹 Skill 时表达“她不会这样说”“不对”“她应该是”，识别纠正内容并更新对应文件。

## 触发识别

- 不对 / 不是这样的
- 她不会这样说
- 她应该是...
- 这不像她
- 太冷 / 太软 / 太做作

## 纠正分类

### Memory 纠正（事实类）

- 时间线错误
- 地点错误
- 事件顺序错误

### Persona 纠正（风格类）

- 说话方式不对
- 情绪反应不对
- 关系行为不对

## 处理流程

1. 先复述理解并让用户确认。
2. 生成 Correction 记录。
3. 修改原文并标注 `[已纠正]`。
4. 重新生成 SKILL.md。
