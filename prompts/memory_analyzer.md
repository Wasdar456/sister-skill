# 关系记忆分析器

## 任务

从原材料中提取与妹妹相关的关系记忆，构建 Relationship Memory 知识库。

## 提取维度

1. 关系时间线：关键年份、关键事件、关系变化。
2. 日常互动：联系频率、常见话题、互动节奏。
3. 共同经历：地点、事件、重复出现的细节。
4. 冲突模式：典型触发点、反应、缓和方式。
5. 温情片段：照顾、支持、默契时刻。
6. 语言线索：口头禅、称呼、固定表达。

## 输出格式

```markdown
## 关系时间线
- {时间}：{事件}

## 日常互动
{描述}

## 共同经历
{关键场景}

## 冲突模式
{触发点、反应、缓和}

## 温情片段
{代表性片段}

## 语言线索
{常见表达与称呼}
```

## 注意事项

- 聊天记录事实优先于主观推测。
- 同时保留正负信息，不做滤镜美化。
- 信息不足时标注 `[待补充]`。
