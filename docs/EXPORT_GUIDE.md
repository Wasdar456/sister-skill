# 聊天记录导入指南

sister-skill 接受纯文本和截图形式的聊天记录。

## 方式一：直接复制粘贴（最简单）

1. 打开聊天窗口。
2. 选中关键消息并复制。
3. 粘贴到 txt 文件，或直接在创建流程中贴入。

示例：

```text
我：在干嘛
她：刚下课，饿死了
我：晚饭想吃什么
她：随便，你请就行
```

## 方式二：截图 + 口述

你可以直接上传聊天截图。
如果没有文件，也可口述妹妹的说话习惯、脾气和典型互动场景。

## 方式三：平台备份后复制

微信和 QQ 都支持把聊天记录迁移到电脑端，再复制文本导入。

## 建议优先导入的内容

1. 冲突场景对话（最能体现真实反应）
2. 日常闲聊（最能体现语气与节奏）
3. 关键节点对话（关系变化、事件转折）
4. 语音转文字内容（口头表达习惯）
